# Commit Message Syntax

Syntax Highlighting for git commit/tag/merge messages (`git commit`, `git tag -a`)

Includes highlighting for:
 - Long Summary Line
 - Text on the second line
 - Comments
 - Long lines
 - Author/Date
 - Branches
 - Status
 - Changes (staged/unstaged/untracked)
 - diff (commit -v)
